# Prerequisites

Before running this project, you must have node.js downloaded and npm downloaded which is included in node.js.
This can be installed at https://nodejs.org/en/download

# Installation

To install this project use git clone. 

`git clone https://github.com/Yusuf-197/Weather-App-Group51.git`

Then using your terminal navigate to the directory using cd.

For example `cd weatherapp`.

Finally install any dependencies using `npm install`

## Get an API Key 

We created our weather application using data fetched from weatherapi.com.

Create an account and generate an api key. 

Then copy your API key into a .env file and store it in a variable called REACT_APP_WEATHER_API_KEY.


## Running the application 

To run the application use `npm start` in your directory terminal.
